# reference/

`prototype-web.jsx` — bu ürünün **web prototipi** (React, tek dosya, sahte verili).
Amacı: hedef UX'i ve ekran akışını göstermek. React Native'e **birebir taşıma** değil,
**referans**tır: ekranlar (home / setup / play / history / session), oran tabanlı
kalem izleri, "satıra bağlama" mantığı ve "tanınan değerler" paneli buradaki gibi olmalı.
Tanıma kısmı prototipte elle onayla temsil edilir; gerçek uygulamada ML Kit doldurur.
